# Dynamic Inventory Management System - Phase 2 Proof of Concept

## Overview
This project is a proof-of-concept implementation of a dynamic inventory management system developed in Python. It demonstrates how data structures such as dictionaries, category mappings, and priority queues can be used to support common inventory operations.

## Features
- Add new products
- Update product quantities
- Search for products
- Retrieve products by category
- Identify low-stock items for restocking
- Delete products
- Run test cases for validation

## Project Files
- `inventory_manager.py` - main class implementation
- `demo_inventory.py` - demonstration script
- `test_inventory.py` - unit tests
- `README.md` - project documentation

```bash
python demo_inventory.py
